# 基本面与股价关系：经济天气预报

## 核心观点

> 「股市是经济的晴雨表」——这句话是错的。

## 真正关系

```
股价 = 不是当前基本面
     = 而是"未来9-12个月的预期基本面"
     = 本质上是"经济天气预报"
```

## 数据验证

### A股历史最长ROE衰退

| 时间 | 全A股ROE中位数 |
|------|---------------|
| 2021年高点 | 9.18% |
| 当前（2025年初） | 4.75% |
| 连续下跌 | **13个季度** |

> 「这是A股历史上持续时间最长、调整幅度最大的一次基本面衰退」

## 关键认知

### 1. 股价领先基本面

股价提前9-12个月反映未来经济预期：
- 经济好 → 股价先涨
- 经济差 → 股价先跌

### 2. 基本面差 ≠ 股价跌

即使基本面很差，只要预期未来会变好，股价也可以涨。

### 3. 基本面好 ≠ 股价涨

即使基本面很好，如果预期未来会变差，股价也可以跌。

## 实战含义

### 不要看当前经济数据做股票

当前经济数据差 = 可能在底部
当前经济数据好 = 可能在顶部

### 关键是看预期

- 预期变好 → 买入
- 预期变差 → 卖出

## 美股 vs A股

### 美股

- 股价领先经济3-6个月
- 更有效率

### A股

- 股价领先经济6-12个月
- 情绪放大波动

## 重要提醒

> 「看当前基本面做股票，往往会买在顶部、卖在底部。」

真正的投资者应该关注：
- 未来预期如何变化
- 政策转向的信号
- 资金面的实际流入流出
