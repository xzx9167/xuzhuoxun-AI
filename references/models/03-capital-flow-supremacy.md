# 模型3：资金面权重最高

## 一句话总结
ETF净申购 / 融资余额 / 超大单净流入是A股核心变量，**其他都是噪声**。

## 核心逻辑
```
资金 = 价格涨跌的真正驱动力
情绪、基本面、政策 → 最终都通过资金体现
```

## 三大资金指标

| 指标 | 代表 | 含义 | 怎么看 |
|------|------|------|-------|
| **ETF净申购** | 机构长期资金 | 机构态度 | 连续净申购 → 看多 |
| **融资余额** | 散户短期情绪 | 杠杆使用 | 快速攀升 → 风险积累 |
| **超大单净流入** | 头部资金 | 真实动向 | 由流出转流入 → 阶段性底部 |

## 经典语录

> 「现在市场是增量还是存量资金？如果是存量资金，那么大盘没有方向，只剩结构性机会。」

> 「ETF净流入=机构长期资金态度；融资余额=散户短期情绪；超大单=头部资金最真实动向。」

## 资金面分析框架

### 增量市场信号
- ETF连续净申购（每日监测）
- 两融余额稳步上升但未疯狂
- 超大单持续净流入
- 北向资金持续买入

### 存量市场信号
- ETF申赎基本持平
- 两融余额区间波动
- 超大单来回反复
- 大盘无趋势，结构性机会

### 减量市场信号
- ETF净赎回
- 融资余额快速下降
- 超大单持续净流出
- 现金为王

## 政策市特点

> 「国家队行为无法预测，有时会干预数据信号。」

**应对**：
- 资金面信号与政策方向一致时，信号更可靠
- 政策强力干预时，技术分析暂时失效

## 核心关注数据（2026年）

- 存款搬家进度（居民存款同比 + 非银存款同比）
- ETF申赎数据（每日）
- 融资余额变化（每日）
