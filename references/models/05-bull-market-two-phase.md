# 模型5：牛市两阶段论

## 一句话总结
第一阶段资金驱动，第二阶段情绪驱动，**存款搬家进度是核心进度条**。

## 核心逻辑
```
牛市不是一次性涨完
而是分阶段推进
每一阶段有不同的驱动力和特征
```

## 两个阶段对比

| 特征 | 第一阶段 | 第二阶段 |
|------|---------|---------|
| 驱动力 | 资金面改善 | 情绪面爆发 |
| 上涨逻辑 | 估值修复 | 预期打满 |
| 参与资金 | 机构/聪明钱 | 散户/杠杆 |
| ETF申购 | 明显净申购 | 疯狂净申购 |
| 两融余额 | 稳步回升 | 快速飙升 |
| 散户参与度 | 观望为主 | 跑步进场 |
| 监管态度 | 呵护 | 警惕 |

## 存款搬家进度

> 「存款搬家进度是牛市最核心的进度条。」

### 观察指标
- 居民存款同比增速（趋势下降=搬家进行中）
- 非银存款同比增速（搬家进行中）
- 证券公司新开户数
- 偏股基金发行规模

### 搬家阶段判断

```
存款搬家初期 → 资金刚起步，机会最多
存款搬家中期 → 情绪开始爆发，波动加大
存款搬家末期 → 杠杆疯狂，离场信号
```

## 经典语录

> 「牛市第一阶段是资金驱动，这时候 ETF 净申购是最重要的信号；牛市第二阶段是情绪驱动，这时候两融余额的增速是最重要的信号。」

> 「判断牛市到哪个阶段，就是看存款搬家到哪个阶段。」

## 当前市场判断框架（2026年）

根据徐老师2026年最新观点：

### 核心观察数据
1. **存款搬家进度**：居民存款是否在向股市搬家
2. **ETF申赎数据**：每日监测机构态度
3. **两融余额变化**：每日监测杠杆使用程度
4. **超大单净流向**：头部资金真实动向

### 当前阶段特征（2026年初）
- 处于牛市第一阶段向第二阶段过渡
- 存款搬家进度持续
- ETF仍有净申购
- 两融余额稳步回升但未疯狂

## 牛市顶部信号

1. 两融余额创历史新高
2. 新基金发行爆款日光
3. 全民讨论股票
4. 存款搬家进度到顶
5. 监管开始提示风险

## 牛市三个阶段（资金流框架）

### 第一阶段：资金停止流出

```
所有看空资金都已卖出
↓
没有抛压了
↓
稍微一点增量资金就能上涨
↓
情绪底部出现
```

**特征**：
- ETF净申购开始
- 超大单由流出转流入
- 成交量萎缩后温和放大

### 第二阶段：增量资金持续入场

```
增量资金持续买入
↓
赚钱效应出现
↓
存款搬家加速
↓
场外资金跑步进场
```

**特征**：
- 存款搬家进度加快
- 两融余额稳步攀升
- 散户开户数增加

### 第三阶段：增量资金枯竭

```
没有更多增量资金了
↓
高估值抛压超过承接能力
↓
牛市结束
```

**判断标准**：
- 存款搬家进度到顶
- 增量资金流入速度放缓
- 头部资金开始撤退

## 牛市结束的三个问题

当市场下跌时，问三个问题：

1. **场外还有没有钱？**
   - 存款搬家是否还在进行
   - 居民存款增速是否极低

2. **场外资金还愿意进来吗？**
   - 赚钱效应是否还在
   - 情绪是否依然亢奋

3. **增量资金能否承接上方抛压？**
   - 成交量是否足够
   - 头部资金是否仍在买入

> 「如果三个问题答案都是YES——牛市还在。如果任何一个开始转向——就要小心了。」

## 历史规律

| 时间 | 存款搬家峰值 | 市场顶部 |
|------|-------------|---------|
| 2014年 | 增速触底回升 | 5178点 |
| 2018年初 | 增速触底回升 | 3587点 |
| 2021年底 | 增速触底回升 | 3731点 |

**规律**：存款增速降到极值后回升 = 牛市阶段性顶部

## 7-10年一轮牛市

> 「中国股市每7-10年一轮牛市——核心原因是场外可投入股市的存款积攒至一定规模 + 市场周期性起伏」
