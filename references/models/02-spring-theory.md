# 模型2：弹簧理论

## 一句话总结
跌到支撑 + 缩量止跌 + 第一根阳线 = **赔率最佳的买点**

## 核心逻辑
```
弹簧压紧 → 蓄力 → 松开
压缩得越狠，弹得越高
```

## 经典语录

> 「弹簧压紧的好处是，你参与进去你的证明和证伪的时间窗口是很短的。如果你对了，那么你就获得了一段趋势，而且你是买在这个趋势的阶段最底部；如果你错了，那你也就是付出一小段止损的钱。」

> 「短炒就像放风筝，就是你在风来的时候才能起飞，如果风停下来了，你得休息。」

## 弹簧买入信号（必须同时满足）

| 条件 | 说明 | 观察指标 |
|------|------|---------|
| 1. 回踩支撑 | 价格跌到关键均线/支撑位 | MA5/MA10/MA20/前期低点 |
| 2. 缩量止跌 | 下跌时成交量萎缩 | 量能萎缩至起涨前30%以下 |
| 3. 第一根阳线 | 止跌后首日收阳 | 日K线颜色、实体大小 |

```
视觉记忆：
     /\    ← 第一根阳线（买点）
    /  \  ← 缩量止跌
   /    \
  /      \  ← 下跌
 /
/        ← 起涨点
```

## 弹簧卖出信号

> 「风停下来了，你得休息」

- 上涨后放量大阴线
- 连续拉升后量能背离
- 突破后快速收回均线下方

## 弱弹簧识别（关键！）

> 「如果弹簧被压紧后松开了，最后反反复复压紧，这恰好说明趋势是不对的。一个上涨的趋势是没有太多回档点的，如果反复回档恰好说明回档点太弱了，需要考虑卖出。」

**弱弹簧特征：**
- 反复压紧、松开、再压紧
- 趋势不流畅
- 每次支撑位都勉强撑住

**应对：**
- 第一次压紧 → 正常仓位
- 第二次压紧 → 减半仓
- 第三次压紧 → 不参与/撤退

## 与海龟交易的区别

| 交易系统 | 特点 | 适应场景 |
|---------|------|---------|
| 弹簧交易 | 顺大势，逆小势 | 震荡市、回调买入 |
| 海龟交易 | 顺大势，顺小势 | 单边趋势市 |

> 「海龟交易的局限：需要轰轰烈烈的大趋势才能获利，震荡市中反复被打脸」

## 模型局限

**失效场景**：
- 震荡市：弹簧反复压缩松开，**频繁假信号**
- 单边市：效果最好

> 「震荡市里弹簧会反复压缩松开，单边市里弹簧理论效果最好。」

## 实战工具箱

### 均线弹簧
- 回踩5日线 + 缩量 + 阳线 → 短线机会
- 回踩20日线 + 缩量 + 阳线 → 中线机会
- 回踩60日线 + 缩量 + 阳线 → 长线机会
