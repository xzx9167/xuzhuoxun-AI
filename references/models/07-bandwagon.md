# 模型7：抱团游戏

## 一句话总结
A股本质是一场抱团游戏——**机构、散户、量化都在抱团，只是抱的对象不同**。

## 核心逻辑

```
市场由资金驱动
资金形成共识
共识推动行情
共识瓦解 → 反转
```

## 四类资金行为

| 资金类型 | 共识形成方式 | 抱团对象 |
|---------|-------------|---------|
| 机构 | 卖方信息链 | 龙头白马 |
| 游资/散户 | 概念题材 | 热点概念 |
| 量化 | 历史规律 | 因子策略 |
| 羊群资金 | 跟涨杀跌 | 来回切换 |

## 抱团的终点

> 「当所有人都在谈论同一只股票/板块时——反转就来了」

### 数据信号

当公募基金单一行业配置达到极端值时：
- 食品饮料占17.6%（2020年Q4）→ 板块见顶
- 医药占18.9%（2020年Q2）→ 板块见顶

### 股东户数

> 「股东户数是一个有效因子——股东太多 = 顶部」

## 抱团的生命周期

```
发酵期 → 共识形成 → 抱团加强 → 顶部信号 → 反转
```

### 发酵期
- 只有少数人开始关注
- 资金悄悄布局
- 股价缓慢上涨

### 共识形成
- 媒体开始报道
- 散户开始买入
- 资金加速流入

### 抱团加强
- 所有人都知道
- 场外资金持续涌入
- 股价快速上涨

### 顶部信号
- 全民讨论
- 基金发行爆款
- 估值泡沫化

### 反转
- 聪明资金开始撤退
- 共识瓦解
- 快速下跌

## 抱团应对策略

> 「你可以喝，甚至可以喝到七分醉，但不能醉倒趴在酒桌上睡着」

### 正确姿势
- 参与抱团行情
- 识别见顶信号
- 在泡沫化前离场
- 不要成为最后一个接棒的人

### 错误姿势
- 相信"这次不一样"
- 越涨越加仓（倒金字塔）
- 忽视见顶信号

## 抱团瓦解后的教训

> 「你可以喝，但不要醉倒趴在酒桌上——你会变成那个买单的人」
