# 模型11：运动战思维

## 一句话总结
**打得赢就打，打不赢就走。关键支撑位买入测试，能撑住就持有，撑不住就走。**

## 核心思想

```
运动战源自毛主席军事思想：
"打得赢就打，打不赢就走"
"你打你的，我打我的"
"集中优势兵力，各个击破"
```

**应用到A股：**
- 买之前先想好什么情况下要跑
- 用止损线测试市场，判断对了持有，判断错了走
- 不追求每次判断都对，追求对的时候赚够、错的时候亏少

## 核心理念

### 1. 不预测，只应对

> 「未来无法预测，只能应对。正确的做法是找到在面对两个平行宇宙中不会受到伤害的解。」

**实战应用：**
- 不猜涨还是跌，而是制定应对计划
- 涨了怎么办？跌了怎么办？
- 两个方向都有预案

### 2. 测试思维

> 「关键支撑位买入测试，能撑住就持有」

**操作流程：**
1. 在支撑位买入
2. 观察能否撑住
3. 撑住 → 持有
4. 撑不住 → 走

### 3. 底线思维

> 「一定要有底线意识，如果这个东西我判断错了，我能接受的最大损失是多少」

**核心问题：**
- 买入前先想好止损位
- 跌破止损无条件走
- 不抱侥幸心理

### 4. 宁可错过，不可做错

> 「趋势钝化后不新开仓位。宁可错过反弹，不可深套其中。」

**实战原则：**
- 错过一天不是大事
- 放个屏幕在边上，隔一段时间看一下
- 趋势钝化后操作的核心逻辑是：宁可错过反弹，不可深套其中

## 弹簧交易法的运动战本质

### 三大规则

| 规则 | 运动战对应 | 说明 |
|------|-----------|------|
| 买入时机 | 试探性进攻 | 在支撑位小仓位测试 |
| 止损退出 | 打不过就跑 | 触及止损无条件走 |
| 弱弹簧识别 | 敌情不对 | 反复压紧=撤退信号 |

### 打得赢就打

**上涨趋势确认后：**
- 持有仓位
- 让利润奔跑
- 趋势不破不出

### 打不赢就走

**触发以下条件时：**
- 跌破支撑
- 趋势钝化
- 止损触发
- 达到目标价

## 与海龟交易的区别

| 交易系统 | 特点 | 适应场景 |
|---------|------|---------|
| 弹簧交易 | 顺大势，逆小势 | 震荡市、回调买入 |
| 海龟交易 | 顺大势，顺小势 | 单边趋势市 |

**海龟交易的局限：**
- 需要轰轰烈烈的大趋势才能获利
- 震荡市中反复被打脸

## 实战应用

### 每日操作流程

```
1. 开盘前：制定今日应对计划
   - 如果涨到XXX，我怎么做？
   - 如果跌到XXX，我怎么做？

2. 盘中：执行计划，不临时决策
   - 到了支撑位 → 测试
   - 趋势正常 → 持有
   - 趋势钝化 → 减仓

3. 收盘后：复盘计划执行情况
   - 判断对了？ → 总结
   - 判断错了？ → 认错改进
```

### 仓位管理

```
第一次压紧 → 正常仓位测试
  ↓
撑住了 → 加仓
  ↓
又压紧 → 减半仓
  ↓
再压紧 → 不参与
```

## 关键原则

1. **不预测**：只制定应对计划
2. **有底线**：先定止损再买入
3. **测试思维**：小仓位试探，确认后再加
4. **跑得快**：判断错了就跑，不侥幸
5. **宁可错过**：趋势钝化后不新开仓
