# 模型1：赔率优先于概率

## 一句话总结
不追求高胜率，追求高赔率，**结构决定命运**。

## 核心逻辑
```
向上空间（赔率） > 向下空间（风险）
→ 出手
→ 不纠结涨的概率是60%还是40%
```

## 经典语录

> 「在概率跟赔率之间，我一般只会考虑赔率。大部分人都没有办法完全把握概率的问题，只要赔率差不多，我就会出手。」

> 「我之前的时候胜率是51%的样子，反正就是比50多一点点。今年因为是牛市，稍微上来了一点，大概也就是53~54%，不会比这个更高了。」

## 实战应用

### 问自己
1. **向上空间是多少？** （目标价 - 买入价）
2. **向下空间是多少？** （买入价 - 止损价）
3. **赔率 = 向上空间 ÷ 向下空间**
4. **赔率 ≥ 2:1** → 考虑出手

### 决策树
```
输入：某标的买入价10元，止损价9元，目标价13元

计算：
- 向上空间 = 13 - 10 = 3元
- 向下空间 = 10 - 9 = 1元
- 赔率 = 3:1

结论：赔率3:1，值得出手
```

## 模型局限

**失效场景**：
- 政策突变（关税突袭）
- 黑天鹅事件
- 流动性枯竭

> 「当赔率结构被政策/黑天鹅完全破坏时，此模型失效。」
