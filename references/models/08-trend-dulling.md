# 趋势钝化与趋势判断

## 核心原则

**价格比趋势更重要。** 判断趋势是否正常的关键是看价格有没有**破位**。

---

## 趋势判断标准

### 趋势正常的标志

- 价格在40天线附近能撑住
- 跌破后能够快速收回来
- 均线呈多头排列
- 温和上涨中逐步吸引人气

### 趋势不正常的标志

- 跌破均线且不能快速收回
- 反复回档
- 波动率异常
- 人气消退

### 支撑位类型（由强到弱）

1. **均线（20日、40日）** - 支撑力度最强，属于短期回调支撑
2. **前期低点** - 中等力度，上一次回调的最低点
3. **震荡区间下沿** - 中等力度，横盘整理的底部
4. **年线/半年线** - 较弱，因变量较多

---

## 趋势钝化

### 什么是趋势钝化

趋势动能开始衰竭，表现为：
- 上涨或下跌动能减弱
- 波动率下降但方向不明
- 可能随时出现反向波动

### 趋势钝化五大信号

| 信号类型 | 具体表现 |
|---------|---------|
| 大阴线/大阳线 | 趋势末端出现反向大波动 |
| 异常反向波动 | 原本强势的板块突然大跌 |
| MACD放大后走弱 | MACD放到较大水平后往下走 |
| 人气消退 | 热点讨论减少，板块不再创新高 |
| 接近压力位 | 价格到达区间上沿或前期高点附近 |
| 波动率收窄 | 横盘但波动率下降，反向波动可能酝酿 |

### 实战案例

**有色金属板块（11月中旬）：**
> 「有色金属板块的周五走势，其实是一个比较明确的趋势钝化信号。首先它的趋势已经没有之前那么流畅了，新的反向趋势开始有些萌芽了。」

- 趋势不再流畅 + 人气消退 + 区间高位 → 减仓一半

**新能源板块：**
- 板块已到后端，情绪空间不大
- 趋势钝化 = 逆转信号 → 减仓

**趋势不再流畅的核心判断：**
- 原本流畅的上涨变得磕磕绊绊
- 每次回调都创新低
- 需要反复测试支撑位

## 趋势交易核心要素

### 弹簧交易法三大规则

1. **规则一：买入时机** - 弹簧压紧时买入，弹簧松开后判断是否减仓
2. **规则二：止损退出** - 弹簧断掉时撤退，触及止损 = 趋势逻辑否定
3. **规则三：弱弹簧识别** - 弹簧反复压紧说明是弱弹簧，需撤退

### 买入时机标准

- 价格跌到均线/支撑位附近
- 波动率收窄
- 缩量
- 市场跌不动

### 仓位管理（递减法则）

- 第一次压紧：正常仓位
- 第二次压紧：减半仓
- 第三次：不参与

### 止损设置方式

- 支撑位止损（如压40天线，40块，跌到39块就止损）
- 跌破均线2%以上清仓
- 时间止损

---

## 趋势反转判断

### 判断趋势即将反转的核心信号

1. **趋势钝化出现时** - 趋势动能衰竭的信号一旦出现，应考虑离场

2. **最好离场位置** - 趋势钝化的那一刹那

3. **次好离场位置** - 等一个反弹再走

4. **趋势钝化后** - 不建议新开仓位

### 核心理念

> "趋势钝化意味着动能衰竭，此时风险收益比急剧恶化。宁可错过反弹，不可深套其中。"

### 实战操作原则

- 错过一天并不是什么太大的事情
- 放个屏幕在边上，隔一段时间看一下
- 趋势钝化后操作的核心逻辑是：宁可错过反弹，不可深套其中

---

## 弹簧交易 vs 海龟交易

| 交易系统 | 特点 |
|---------|------|
| 弹簧交易 | 顺大势，逆小势 |
| 海龟交易 | 顺大势，顺小势 |
| 海龟局限 | 需要轰轰烈烈的大趋势才能获利 |
