# 弹簧理论篇

关于弹簧买入法的经典语录。

## 核心理念

> 「弹簧压紧的好处是，你参与进去你的证明和证伪的时间窗口是很短的。如果你对了，那么你就获得了一段趋势，而且你是买在这个趋势的阶段最底部；如果你错了，那你也就是付出一小段止损的钱。」

> 「弹簧压缩得越狠，弹起来越高。」

## 操作要点

> 「弹簧买入的三个条件：回踩支撑、缩量止跌、第一根阳线。缺一不可。」

> 「回踩5日线 + 缩量 + 阳线 → 短线机会；回踩20日线 + 缩量 + 阳线 → 中线机会；回踩60日线 + 缩量 + 阳线 → 长线机会。」

## 卖出时机

> 「风停下来了，你得休息。」

> 「涨上去之后放量大阴线，就要考虑走了。」

## 市场特点

> 「震荡市里弹簧会反复压缩松开，单边市里弹簧理论效果最好。」

> 「弹簧理论在单边市里效果最好，在震荡市里假信号多。」
