# 三种能力：你需要哪个？

## 核心问题

> 「普通人想在股市赚钱，需要具备哪三种能力？」

| 能力 | 说明 |
|------|------|
| 交易能力 | 减少财富再分配损耗 |
| 选股能力 | 捕捉马太效应 |
| 择时能力 | 买在低位，卖在高位 |

## 数据真相

### 交易能力

| 交易频率 | 3年回报 |
|---------|--------|
| 每天10次以上 | **-79%** |
| 每月1次 | **+59%** |

### 选股能力

> 「A股前0.35%股票创造100%净财富——没买到这些就跑不赢指数」

### 择时能力

> 「85%散户在牛市下半场入场——倒金字塔加仓」

## 财富再分配数据

**10000名投资者，真实A股数据：**

| 结果 | 比例 |
|------|------|
| 亏损 | **79%** |
| 盈利 | 21% |
| 跑赢沪深300 | **3%** |

> 「一将功成万骨枯」

## 三个能力的悖论

### 1. 三个能力都具备

→ 可以自主交易，精选个股，精准择时

### 2. 缺少任何一个

→ 建议：买指数基金

## 指数基金的魔力

| 优势 | 说明 |
|------|------|
| 减少交易 | 避免财富再分配 |
| 分散投资 | 捕获马太效应 |
| 长期持有 | 放弃择时 |

> 「满招损，谦受益——有时候放弃反而是得到」

## 巴菲特的赌约

- 10年期赌约：S&P500指数 vs 5只对冲基金
- 结果：指数完胜
- 原因：
  1. 市场足够有效，90%+专业投资者跑不赢指数
  2. 交易成本侵蚀收益

## 普通人的最优解

```
没有三种能力 → 买宽基ETF（A500/沪深300）
有交易能力 → 可以短线操作
有选股能力 → 精选个股
有择时能力 → 精准进出
三种都有 → 可以考虑主动管理
```

## 关键认知

> 「承认自己不具备某种能力，比假装具备要聪明得多」

> 「对于没有三种能力的普通人，被动指数投资是最好的选择」
