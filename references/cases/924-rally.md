# 924行情：政策反转与资金驱动

## 背景

2024年9月24日，金融支持经济高质量发展新闻发布会宣布一系列重磅政策，开启了新一轮牛市行情。

## 市场环境

- 前期市场持续低迷，指数处于低位
- 政策出现重大转向
- 存款搬家进度处于低位
- 场外资金观望情绪浓厚

## 行情特征

### 第一阶段：政策驱动（9-10月）

| 特征 | 描述 |
|------|------|
| 驱动力 | 政策转向 + 资金入场 |
| ETF申购 | 大幅净申购 |
| 超大单 | 924当周净流入602亿元 |
| 市场情绪 | 从极度悲观快速转向乐观 |

### 第二阶段：情绪爆发（10月后）

| 特征 | 描述 |
|------|------|
| 驱动力 | 情绪 + 增量资金 |
| 两融余额 | 快速攀升 |
| 存款搬家 | 加速进行 |
| 市场特征 | 赚钱效应扩散 |

## 关键数据

- 924当周超大单净流入：602亿元
- 2025年1月13日当周超大单净流入：43亿元（元旦暴跌后阶段性见底）
- 2025年6月23日当周超大单净流入：128亿元（随后开启主升浪）
- 924行情以来每次超大单明显流入后，往往是阶段性启动点

## 头部资金行为分析

> 「每次来到压力位置的时候，都有头部资金抢跑式卖出。从图中可以很明显看出这一点，在主升浪开启之前，千亿级别的超大单流出就会把市场带下来。」

> 「本周的重要会议期间，实际上头部资金并没有卖出，且在会议结束后，这些头部资金也没有抢跑式卖出。本周的超大单居然是流入的，全周流入了109亿元。这反映头部资金居然认为这里是一个加仓点。」

## 超大单与行情关系

| 时间点 | 超大单流向 | 市场表现 |
|--------|-----------|---------|
| 924当周 | 净流入602亿 | 行情启动 |
| 1月13日当周 | 净流入43亿 | 阶段性见底 |
| 6月23日当周 | 净流入128亿 | 开启主升浪 |
| 12月17日 | A500ETF集中净申购 | 双底结构确认 |

## 复盘要点

**成功要素**：
- 政策转向是行情启动的催化剂
- 超大单流向是判断头部资金态度的有效指标
- 存款搬家进度决定了行情能走多远

**经验**：
- 924行情如果对标历史上2005年和2013年，2024年对应的是市场刚起步的阶段
- 行情初期往往伴随技术指标失效，因为价格没有反映一切信息
- 慢思考和长逻辑是大部分人不愿意做的，但愿意静下心来思考会获得极大优势
