# 弹簧理论买点复盘

## 背景

某标的从12元高点下跌至10元支撑位附近，成交量持续萎缩。

## 信号识别

### 1. 回踩支撑
- 价格跌至20日均线（10元）
- 前期低点支撑位

### 2. 缩量止跌
- 成交量萎缩至起涨前的20%
- 量能创近期新低

### 3. 第一根阳线
- 止跌后次日收阳
- 实体较大，突破5日均线

## 决策过程

```
买入价：10.2元
止损价：9.8元（支撑位下方）
止损幅度：约4%

赔率计算：
向上空间：11.5元（前期高点）- 10.2元 = 1.3元
向下空间：10.2元 - 9.8元 = 0.4元
赔率：3.25:1
```

## 结果

- 买入后价格上涨至11.5元
- 盈利约13%
- 持有约2周

## 复盘

**成功要素**：
- 三个条件同时满足
- 止损位设置合理
- 赔率超过3:1

**风险控制**：
- 严格执行止损
- 次日开盘跌破支撑，无条件清仓
