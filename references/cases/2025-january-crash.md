# 2025年1月开年暴跌

## 背景

2025年1月初，市场出现开年暴跌行情，大盘指数大幅下跌。

## 市场环境

- 前期涨幅较大
- 市场情绪开始恐慌
- 资金面出现紧张迹象

## 信号识别

### 资金面信号
- ETF申赎开始出现分化
- 两融余额开始下降
- 超大单出现净流出

### 技术信号
- 大盘指数跌破20日均线
- 成交量放大（恐慌抛售）
- 多个板块同步下跌

## 弹簧信号

在暴跌过程中，部分优质标的出现了弹簧信号：

```
标的A：
- 回踩60日均线（长线支撑）
- 量能萎缩至前期30%
- 止跌后收出第一根阳线
```

## 决策与执行

### 仓位管理
- 以损定量确定仓位
- 止损位设在60日均线下方5%
- 单笔最大亏损控制在2%

### 结果
- 标的A在弹簧买入后上涨约20%
- 其他未出现弹簧信号的标的未参与

## 复盘

**经验**：
- 暴跌中也有机会，关键看是否出现弹簧信号
- 仓位管理到位，所以能够承受波动
- 没有弹簧信号的标的绝不强行买入

**教训**：
- 暴跌初期不应抄底
- 等待弹簧信号出现后再入场
- 止损纪律必须严格执行
