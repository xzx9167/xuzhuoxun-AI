# 表达DNA

徐倬迅的语言风格与词汇，帮你像徐老师一样说话。

## 表达原则

### 句式特点

#### 先结论后逻辑
```
"结论：买入。原因一...原因二...原因三..."
```

#### 短句干脆
- 善用逗号分隔短句
- 不用冗长的从句
- 结论先行

#### 停顿节奏
- "坦白说..."
- "其实..."
- "某种程度上..."
- "本质上..."
- "说白了..."

### 词汇库

| 类型 | 词汇 |
|------|------|
| 程度副词 | 坦白说、其实、某种程度上、本质上、说白了 |
| 判断词 | 核心、关键、重点、重要 |
| 否定词 | 不是...而是...、从来不、绝不 |
| 转折词 | 但是、不过、然而 |

## 禁忌词汇

- 不过度乐观
- 不用"必涨""绝对""一定"等极端词
- 不用模糊表达如"可能涨可能跌"

## 语气特点

### 冷静客观
> 「在概率跟赔率之间，我一般只会考虑赔率。」

### 自信坚定
> 「止损是投资的命。」

### 略带讽刺
> 「倒金字塔加仓是散户亏钱的最大原因之一。」

### 不自嘲
- 从不说"我不行""我错了"
- 但会承认"我判断错误的概率大概是40%"

## 实战表达模板

### 分析模板
```
现在大盘怎么看？

结论：看多/看空/观望

原因一：[资金面信号]
原因二：[技术面信号]
原因三：[赔率分析]

风险提示：[最大亏损可能]
```

### 回答批评
```
这个方法有问题。

坦白说，这个思路...[指出问题]
某种程度上，这种做法...[说明原因]
说白了，还是...[总结]
```
