# 交易心理学

从"心魔"到"圣杯"的实战心法。

## 三大心魔

### 1. 损失厌恶

**表现**：
- 亏钱的痛苦是赚钱快乐的2倍
- 卖出盈利股票太快
- 持有亏损股票太久

**数据**：
```
60%胜率，+10%赚/-30%亏
= -6%预期回报（亏损）
```

**期望推导**：
- 60% × (+10%) + 40% × (-30%) = 6% - 12% = -6%
- 每交易一次，预期亏损6%
- 交易越频繁，亏损越多

**解决**：
- 用止损线控制损失
- 用赔率代替概率思考
- 亏小钱不亏大钱

### 2. 频繁交易

**表现**：
- 每天盯盘
- 抓住每一个机会
- 交易成瘾

**数据**：
- 每天交易10次以上：3年回报 **-79%**
- 每月交易1次：3年回报 **+59%**

**解决**：
> 「宁可错过机会，也不冒无用的风险」

- 设置"止损假期"
- 减少无效交易

### 3. 过度自信

**表现**：
- 把运气当成能力
- 守株待兔
- 忽视风险

**小数定律陷阱**（特沃斯基/卡尼曼研究）：
> 「把几次运气当成能力重复使用——这就是小数定律」

- 连续盈利3次 → 觉得找到了圣杯
- 买对一次股票 → 以为是实力
- 小样本成功 → 重复使用相同策略

**解决**：
- 定期复盘
- 区分运气和能力
- 相信"市场永远正确"

## 高胜率陷阱

> 「世界上确实有超高胜率策略，但往往都存在缺陷——压路机前捡钢镚」

### 卖期权的高胜率陷阱

**案例**：James Cordier 2018年天然气期权爆仓
- 卖期权策略胜率高达90%+
- 遇到黑天鹅直接破产
- 高胜率不代表低风险

**教训**：
- 不能只看胜率，要看赔率分布
- 黑天鹅事件一次就可能归零
- 高胜率 + 低赔率 = 慢性死亡

## 核心原则

### 1. 逆人性

> 「买在别人恐惧时，卖在别人贪婪时」

> 「截断亏损，让利润奔跑」

> 「大部分散户做的恰恰相反：卖出赢利的，持有亏损的」

### 2. 决断力

> 「计划你的交易，交易你的计划」

> 「机会稍纵即逝——需要肌肉记忆」

### 3. 买者自负

> 「市场永远正确——理解它，不要对抗它」

### 4. 不要侥幸

> 「把运气当能力 = 守株待兔」

### 5. 求真改进

> 「接受错误才能进步」

## 史记名言

> 「顺，不妄喜；逆，不惶馁；安，不奢逸；危，不惊惧；胸有惊雷而面如平湖者，可拜上将军」

## 复盘方法

- 每日：是否执行计划
- 每周：交易频率是否过高
- 每月：亏损来源是什么
- 每季：运气还是能力
