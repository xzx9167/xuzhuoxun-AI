# 人物时间线

关键节点与思想演变，理解徐倬迅的成长脉络。

## 早年经历

- **价值投资起步**：早期做价值投资，研究基本面
- **发现A股特性**：发现A股市场以情绪博弈为主，与价值投资理论有差异

## 2015年股灾

- **股灾冲击**：2015年股灾让杠杆认知刻骨铭心
- **思想转变**：从纯价值投资转向趋势交易
- **核心纪律形成**：
  - 止损纪律
  - 仓位管理原则

## 体系形成

- **赔率优先**：形成赔率优先于概率的核心思维
- **弹簧理论**：总结出弹簧压紧松开的买入方法
- **资金面体系**：建立以资金面为核心的分析框架

## 2025-2026年

- **公众号输出**：通过公众号「徐倬迅的投资日记」分享实战经验
- **知识星球**：在知识星球回答投资者问题
- **知识库整理**：将多年实战经验系统化整理

## 思想演变

```
价值投资 → 趋势交易 → 赔率优先 → 系统化输出
```

## 核心认知转变

| 阶段 | 认知 |
|------|------|
| 早期 | 价值决定价格 |
| 股灾后 | 情绪也能决定价格 |
| 现在 | 资金是最终驱动力，情绪是放大器 |
