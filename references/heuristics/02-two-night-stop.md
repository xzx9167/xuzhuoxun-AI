# 启发式2：两个隔夜止损

## 一句话总结
收盘跌破支撑位，**必须次日开盘清仓**，不过夜。

## 核心逻辑
```
日内波动无法预测
收盘价是当天最终答案
跌破支撑 = 当天判断错误
次日开盘 = 最后的纠错机会
```

## 操作规则

### 触发条件
- 收盘价跌破止损线/支撑位
- 收盘前未收复支撑位

### 执行动作
```
次日开盘 → 无条件清仓
不管开盘价高低
不管后续涨跌预期
```

## 经典语录

> 「止损的原则很简单：如果你当天收盘跌破支撑，那第二天开盘就是你的最后纠错机会。不要等，不要看，不要犹豫。」

> 「很多人亏大钱不是因为买错了，而是因为不止损。跌了觉得会涨回来，结果越亏越多。」

## 为什么是"两个隔夜"

```
第一夜：确认跌破（收盘价确认）
第二夜：执行止损（次日开盘执行）
```

## 关键原则

- **收盘价是判断标准**：不是日内高低点
- **不过夜**：跌破支撑的仓位，持有过夜就是赌博
- **次日必须出**：不管什么价位开盘都要出

## 常见错误

| 错误做法 | 正确做法 |
|---------|---------|
| 觉得会涨回来，持仓观望 | 收盘跌破，次日开盘出 |
| 等反弹到更高再出 | 无条件次日开盘清仓 |
| 分批卖出 | 第一时间全部清仓 |
