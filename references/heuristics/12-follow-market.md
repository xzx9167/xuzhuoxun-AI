# 启发式12：买者自负

## 一句话总结
**市场永远正确，理解它不要对抗它**。

## 核心逻辑

```
市场走势 ≠ 我认为的走势
→ 市场永远正确
→ 理解市场，不要对抗
→ 做市场的跟随者
```

## 经典语录

> 「买者自负，这个市场永远正确，你要理解它，不要对抗它。」

> 「很多人亏钱是因为觉得自己比市场聪明，觉得市场错了他想纠正市场，结果被市场教训。」

> 「当你的判断和市场走势不一致时，一定是你的判断有问题。」

## 实战应用

### 错误做法

| 错误 | 结果 |
|------|------|
| 觉得跌多了就去抄底 | 往往抄在半山腰 |
| 觉得涨多了就卖 | 往往卖完继续涨 |
| 觉得估值低就买 | 可能更低 |
| 觉得消息利好就买 | 可能高开低走 |

### 正确做法

```
1. 市场走势是第一信号
2. 不要试图预测顶底
3. 顺势而为
4. 错了就认，及时纠错
```

## 与市场对抗的代价

> 「和市场对抗的人，最终都会被市场消灭。」

**典型案例：**
- 2015年股灾前：很多人觉得估值太高了要做空，结果继续涨
- 2020年3月：觉得要金融危机，结果反弹
- 2024年：觉得要跌破2600，结果924行情

## 买者自负的含义

1. **为自己的决策负责**：不要怪市场、怪消息、怪机构
2. **承认错误**：判断错了就认，不要侥幸
3. **尊重市场**：市场永远有它的道理
4. **持续学习**：从错误中吸取教训

## 心态修炼

> 「你要承认你有40%多的概率是错的，所以大部分时候你是在认错。这个是正常的。」

**健康心态：**
- 不追求每判断都对
- 追求对的时候赚够、错的时候亏少
- 接受认错是交易的一部分
- 不和市场赌气

## 关键原则

1. **市场永远正确**：走势说明一切
2. **不要对抗市场**：顺势而为
3. **错了就认**：止损是纠错，不是失败
4. **持续改进**：每次错误都是学习机会
