# 启发式4：预备队思维

## 一句话总结
局势不确定的时候，**留20%现金当预备队**。

## 核心逻辑
```
市场永远有机会
现金永远有价值
满仓的人没有选择权
留预备队的人才有选择权
```

## 经典语录

> 「局势不确定的时候，我会留20%左右的现金。这是我应对不确定性的方式。」

> 「预备队是用来应对两种情况的：一是判断错误时的加仓机会，二是市场出现极端机会时的抄底能力。」

## 预备队的作用

### 1. 纠错加仓
```
买入后下跌
在更低价位用预备队补仓
降低成本
```

### 2. 极端机会
```
市场出现极端恐慌
可以用预备队抄底
获取超额收益
```

### 3. 心理缓冲
```
满仓时波动不影响心态
有预备队时心态更稳
```

## 仓位分配参考

| 市场状态 | 仓位 | 预备队 |
|---------|------|-------|
| 极度恐慌/底部 | 80% | 20% |
| 正常/震荡 | 60-70% | 30-40% |
| 极度狂热/顶部 | 50%以下 | 50%以上 |

## 关键原则

- **局势不确定时留20%**：给自己留应对选择的余地
- **预备队不是用来消费的**：只在极端机会时使用
- **根据市场状态调整预备队比例**：底部少预留，顶部多预留
