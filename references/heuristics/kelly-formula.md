# 凯利公式：仓位管理的圣杯

## 核心逻辑

```
f = p - (1-p)/b

f = 最优仓位比例
p = 胜率
b = 赔率（赢损比）
```

## 简化版（赔率=1:1时）

```
f = 2p - 1
```

**60%胜率 → 最优仓位20%**

## 实战案例

### 硬币猜大小实验

**实验设置**：100人初始100元，满仓猜硬币正反（赢+100%/输-50%）

| 策略 | 10万次后结果 |
|------|-------------|
| 满仓(100%) | **100人全部归零** |
| 150%仓位 | 必归零 |
| 凯利(50%) | 长期不赚不亏 |
| 半凯利(25%) | 最优 |

> 「100人初始100元，满仓猜硬币，10万次后中位数打平；150%仓位，100人全部归零。」

### 真实市场数据

- 60%胜率 → 凯利公式最优仓位20%
- 超过2倍凯利 → **必然亏损**
- 1.7倍凯利 → **几乎必然破产**

## 凯利公式的启发

### 1. 仓位管理比选股重要

```
满仓+选错 = 灾难
轻仓+选对 = 稳健
```

### 2. 胜率不够，仓位来补

- 高胜率 → 可以适当加重仓
- 低胜率 → 必须轻仓
- 没有胜率 → 不交易

### 3. 防止过度自信

| 声称胜率 | 实际胜率 |
|---------|---------|
| 80% | 60% |
| 90% | 65% |

> 「自我欺骗导致过度下注 → 必然毁灭」

## 巴菲特的杠杆

- 使用保险浮存金作为永续杠杆
- 可随时调用，有大量现金备用
- 不是制度化杠杆（1.7倍临界值）

## 凯利公式警告

- **过犹不及**：超过最优仓位反而有害
- **历史胜率**：用自己的历史数据计算
- **分散风险**：不要all in单个股票
