# 启发式5：弹簧压紧松开

## 一句话总结
回踩支撑 + 缩量止跌 + 第一根阳线 = **最佳买入点**。

## 核心逻辑
```
弹簧压得越紧，松开的力度越大
股票也是一样
跌到支撑 + 缩量 = 抛压减少
第一根阳线 = 多方开始反攻
```

## 操作信号（三个条件必须同时满足）

### 条件1：回踩支撑
- 价格跌到关键均线（MA5/MA10/MA20/MA60）
- 或者回踩前期低点
- 或者回踩趋势线

### 条件2：缩量止跌
- 成交量萎缩至起涨前的30%以下
- 表示卖压衰竭
- 没有大量抛盘

### 条件3：第一根阳线
- 止跌后首日收阳
- 实体越大越好
- 量能温和放大

## 视觉记忆

```
     /\    ← 第一根阳线（买点）
    /  \  ← 缩量止跌
   /    \
  /      \  ← 下跌
 /
/        ← 起涨点
```

## 经典语录

> 「弹簧压紧的好处是，你参与进去你的证明和证伪的时间窗口是很短的。如果你对了，那么你就获得了一段趋势，而且你是买在这个趋势的阶段最底部；如果你错了，那你也就是付出一小段止损的钱。」

## 均线弹簧级别

| 弹簧级别 | 均线 | 持仓周期 | 止损幅度 |
|---------|------|---------|---------|
| 短线弹簧 | MA5 | 3-5天 | 2-3% |
| 中线弹簧 | MA20 | 2-4周 | 5-7% |
| 长线弹簧 | MA60 | 1-3月 | 10-15% |

## 关键原则

- **三个条件缺一不可**：单独满足任一条件都不是买入信号
- **短线弹簧更容易把握**：但止损要更严格执行
- **止损位置**：在缩量的最低点下方

## 弱弹簧识别（关键！）

> 「如果弹簧被压紧后松开了，最后反反复复压紧，这恰好说明趋势是不对的。一个上涨的趋势是没有太多回档点的，如果反复回档恰好说明回档点太弱了，需要考虑卖出。」

**弱弹簧特征：**
- 反复压紧、松开、再压紧
- 趋势不流畅
- 每次支撑位都勉强撑住

## 仓位递减法则

| 次数 | 情况 | 操作 |
|------|------|------|
| 第一次 | 弹簧正常压紧 | 正常仓位 |
| 第二次 | 弹簧再次压紧 | 减半仓 |
| 第三次 | 弹簧第三次压紧 | 不参与/撤退 |
