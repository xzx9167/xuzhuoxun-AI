# 启发式6：热点生命周期

## 一句话总结
发酵期可买，横盘期不可买，**消散期绝对不能买**。

## 核心逻辑
```
每个热点都有自己的生命周期
在不同阶段买，结果完全不同
—— 买在对的阶段是暴利，买在错的阶段是接盘
```

## 三个阶段

### 阶段1：发酵期（可买）

| 特征 | 说明 |
|------|------|
| 事件刚发生 | 热点刚被点燃 |
| 资金涌入 | 量能开始放大 |
| 龙头股涨停 | 连续涨停打开空间 |
| 媒体开始报道 | 热度扩散中 |

**操作**：积极买入，尤其是龙头股

### 阶段2：横盘期（不可买）

| 特征 | 说明 |
|------|------|
| 龙头高位震荡 | 不再连板 |
| 概念扩散 | 边缘股开始补涨 |
| 换手率高 | 资金在博弈 |
| 热度还在 | 但已经开始分化 |

**操作**：观望，不追高

### 阶段3：消散期（绝对不能买）

| 特征 | 说明 |
|------|------|
| 龙头股下跌 | 开始破位 |
| 概念股普跌 | 鸡犬升天后鸡犬落 |
| 成交量萎缩 | 资金开始撤退 |
| 媒体冷却 | 无人谈论 |

**操作**：卖出，不抄底

## 经典语录

> 「热点 lifecycle 的核心是：龙头不倒，行情不了。龙头开始跌了，这个热点就结束了。」

> 「散户亏钱最多的时候就是在横盘期和消散期。因为他们觉得便宜，觉得还能涨，结果买在最高点。」

## 判断方法

### 龙头股走势
```
发酵期：连续涨停
横盘期：高位震荡
消散期：破位下跌
```

### 成交量变化
```
发酵期：量能持续放大
横盘期：量能高位徘徊
消散期：量能萎缩
```

## 关键原则

- **只买发酵期**：买在热点刚开始的时候
- **横盘期不追**：宁可错过，不可做错
- **消散期必须走**：不抄底，不格局
