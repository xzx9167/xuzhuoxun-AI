# 启发式1：以损定量

## 一句话总结
先定止损线，再倒推仓位，**让止损幅度决定你能买多少**。

## 核心逻辑
```
不是你想买多少，而是你能承受多少损失
根据止损幅度反推仓位
—— 亏损已知，盈利交给市场
```

## 操作步骤

### 步骤1：确定止损线
- 支撑位下方
- 前期低点
- 均线位置

### 步骤2：计算止损幅度
```
止损幅度 = 买入价 - 止损价
```

### 步骤3：确定单笔最大损失
```
单笔最大损失 = 总资金 × 2%
```

### 步骤4：计算仓位
```
仓位 = 单笔最大损失 ÷ 止损幅度
```

## 案例

```
总资金：100万
单笔最大损失：2万（2%）
买入价：10元
止损价：9.5元（5%止损幅度）

仓位 = 2万 ÷ 0.5元 = 4万股 = 40%仓位
```

## 经典语录

> 「我从来不先想能买多少股。我先想的是如果我错了，我最多亏多少。然后根据这个来倒推仓位。」

> 「以损定量是我最重要的仓位管理原则。亏损是成本，赚钱是结果。」

## 关键原则

- **单笔不超过2%**：避免单次错误消耗过大
- **止损幅度要合理**：太小容易止损太多，太大仓位太轻
- **严格执行**：到了止损位必须出
