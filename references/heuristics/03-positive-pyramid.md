# 启发式3：正金字塔仓位

## 一句话总结
越涨越减，**绝不能倒金字塔加仓**。

## 核心逻辑
```
赚钱的时候仓位要越来越轻
亏钱的时候仓位要越来越重
—— 只有这样才能保证赚的时候赚得多，亏的时候亏得少
```

## 正金字塔 vs 倒金字塔

### 正金字塔（正确）
```
买入：10元 → 1万股
涨到11元 → 减到0.8万股
涨到12元 → 再减到0.5万股
```

### 倒金字塔（错误）
```
买入：10元 → 0.5万股
涨到11元 → 加到0.8万股
涨到12元 → 加到1万股
```

## 经典语录

> 「倒金字塔加仓是散户亏钱的最大原因之一。涨的时候仓位轻，跌的时候仓位重，这怎么赚钱？」

> 「正金字塔的核心是：让利润奔跑，但仓位要收缩。因为越涨越危险。」

## 操作原则

### 减仓时机
- 价格上涨后分批减仓
- 每次上涨都适当减仓
- 保留底仓但不超过20%

### 仓位配置
```
成本线附近：仓位较重
盈利30%：仓位减半
盈利50%+：仓位降到20%以下
```

## 关键原则

- **上涨减仓，下跌不加仓**：这是反本能的
- **盈利时仓位递减**：锁住利润
- **绝不追高加仓**：宁可错过，不可做错
